元兒～的 git 設定檔
===
## 使用方法
1. 將`gitconfig`複製到你的家目錄，並改名為`.gitconfig`
2. 將`gitignore_global`複製到你的家目錄，並改名為`.gitignore_global`
3. 設定你的名字和Email

        $ git config --global user.name "<你的名字>"
        $ git config --global user.email <你的Email地址>
4. 開心地用吧～！！
